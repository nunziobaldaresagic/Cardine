# Azure OpenAI come provider LLM

Cardine usa Azure OpenAI (GPT-4o) invece di OpenAI diretto. Il prodotto è B2B enterprise con clienti EU: i CV e i dati di carriera dei dipendenti sono dati sensibili soggetti a GDPR. Azure OpenAI garantisce data residency EU, contratti enterprise con clausole di data processing conformi al GDPR, e SLA di produzione — elementi non disponibili con l'API consumer di OpenAI. I modelli sono identici; cambia solo il piano contrattuale e l'infrastruttura di hosting.
