# Style map

Use these AGIC styles by name (or styleId) instead of setting fonts/sizes manually.

Preferred paragraph styles (template names):
- Normal text: `AGIC corpo` (styleId: `AGICcorpo`)
- Section title: `AGIC Titolo Paragrafo` (styleId: `AGICTitoloParagrafo`)
- Secondary title: `AGIC Titolo 2` (styleId: `AGICTitolo2`)
- List bullet (generic): `AM Elenco puntato` (styleId: `AMElencopuntato`)

When in doubt, consult `.agents/skills/agic-docx/references/template-styles.json`.
